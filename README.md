# MEDOFOOT – Premium Comfort Footwear

MEDOFOOT is a modern footwear brand website focused on orthopedic, diabetic, and comfort footwear. The website is designed to provide a premium shopping experience while allowing customers to place orders directly through WhatsApp.

## Project Goal

Build a fast, responsive, and modern footwear website inspired by leading D2C footwear brands.

Current focus:

* Premium user interface
* Product showcase
* WhatsApp ordering
* Mobile-first experience
* Brand building

Future upgrades:

* Product management dashboard
* User accounts
* Payment gateway integration
* Order tracking
* Inventory management

Run development server:

npm run dev

Build production version:

npm run build

Preview production build:

npm run preview
